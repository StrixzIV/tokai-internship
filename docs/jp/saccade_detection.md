# サッカード検出スクリプトの使い方

## 概要
`detect_saccades.py` は、EEGデータセット（EEGLABの `.set`/`.fdt` 形式）内のEOGチャンネルからサッカード（急速眼球運動）を自動的に検出するスクリプトです。速度の閾値処理と厳密なノイズ除去ゲートを適用し、検出されたイベントをEEGLABに再度追加するためのMATLABスクリプトを出力します。

## 処理の流れ

```mermaid
graph TD
    A([開始]) --> B["EEGLABデータセットの読み込み (.set/.fdt)"]
    B --> C1["EOGチャンネルの分離とフィルタ処理 (0.5 - 30 Hz)"]
    B --> C2["EEGチャンネルの分離とフィルタ処理 (0.5 - 30 Hz)"]
    C1 --> D["EOG速度の計算 (一次微分)"]
    D --> E["速度閾値の算出 (平均値 + Z * 標準偏差)"]
    E --> F["サッカード候補の検出 (find_peaks)"]
    
    F --> G["各サッカード候補についてループ処理"]
    
    G --> H1{"ゲート 1: 速度の外れ値チェック<br>(速度 > 中央値 + 3 * IQR)""}
    H1 -- "はい" --> REJ["候補を除外"]
    H1 -- "いいえ" --> H2{"ゲート 2: マイクロサッカード判定<br>(EOGステップ幅 < 130 µV)"}
    
    H2 -- "はい" --> REJ
    H2 -- "いいえ" --> H3{"ゲート 3: EEGノイズ重複判定<br>(EEG Peak-to-Peak > 200 µV)"}
    
    H3 -- "はい" --> REJ
    H3 -- "いいえ" --> H4{"ゲート 4: 単一ピーク波形判定<br>(±100msのプロミネンス > 40%)"}
    
    H4 -- "はい" --> REJ
    H4 -- "いいえ" --> H5{"ゲート 5: 注視安定性判定<br>(サッカード後EOG標準偏差 >= 全体標準偏差 * 1.5)"}
    
    H5 -- "はい" --> REJ
    H5 -- "いいえ" --> H6{"ゲート 6: 波形の単調性判定<br>(サッカード後速度の符号反転 > 4回)"}
    
    H6 -- "はい" --> REJ
    H6 -- "いいえ" --> PASS["すべての品質ゲートを通過"]
    
    PASS --> I["サッカードオンセット（開始点）の特定<br>(速度ピークから過去へ遡り、5%以下に下がる点)"]
    I --> J["サンプルのレイテンシ変換<br>(秒数 * EEG.srate)"]
    J --> K["MATLABスクリプトの出力生成"]
    K --> L([終了])
```

## 必要条件
必要なライブラリがインストールされていることを確認してください。以下のコマンドでインストールできます。
```bash
pip install mne numpy scipy
```
*(または `pip install -r src/python/phantom-array-experiment/requirements.txt` を使用)*

## 使い方
スクリプトはコマンドラインから実行し、入力パスと出力パスを対話形式で入力します。

### スクリプトの実行
Pythonを使用してスクリプトを実行します。
```bash
python src/python/phantom-array-experiment/detect_saccades.py
```

### 入力項目
プロンプトが表示されたら、以下の情報を入力してください。
1. **Data file path (データファイルのパス)**: 入力するデータセットのパスを入力します。例: `data/80O.set`
2. **Output path (出力パス)**: 出力されるMATLABスクリプトの保存先パスを入力します。例: `out/add_saccade_events.m`

### 出力結果
このスクリプトは、検出されたサッカードをEEGLABで読み込まれた `EEG` 変数にイベントとして追加するMATLABコマンドが含まれた `.m` ファイルを生成します。

MATLABで出力を利用する手順:
1. EEGLABで該当する `.set` ファイルを読み込みます。
2. 生成されたMATLABスクリプトをMATLABエディタで開きます。
3. スクリプトを実行します。これにより、サッカードイベントが `EEG.event` に追加され、整合性のチェックが行われます。

## 技術的な構成とパラメータ

サッカード検出パイプラインの動作は、以下のカスタマイズ可能なパラメータと閾値によって制御されます。

| パラメータ / 設定項目 | コード内の識別子 / コンテキスト | デフォルト値 | 説明 |
| :--- | :--- | :--- | :--- |
| **速度閾値のZスコア** | `threshold_z` | `2.0` | 速度の閾値を決定するために、平均絶対速度に乗算するZスコアの係数。 |
| **最小サッカード間隔** | `min_distance_ms` | `250 ms` | 同一サッカードの二重検出を防ぐために設定される、隣接するサッカード間の最小時間間隔。 |
| **EEGノイズ重複閾値** | `MAX_EEG_P2P_THRESHOLD` | `200 µV` (`0.000200` V) | サッカード検出ウインドウ内に許容されるEEGチャンネルの最大Peak-to-Peak振幅。 |
| **マイクロサッカード除外振幅** | `step_amp` | `130 µV` (`0.000130` V) | 視線追従のサッカードとして分類するために必要な、最小限 EOGステップ電圧変化量。 |
| **注視安定ウインドウ分散** | `post_eog_window` | `< 全体標準偏差の1.5倍` | サッカード後の安定期ウインドウ内で許容されるEOG信号の最大標準偏差。 |
| **単調性チェック符号反転回数** | `sign_changes` | `<= 4回` | サッカード減速期の傾き（スロープ）において許容されるEOG速度の符号反転（方向変化）の最大回数。 |

## 品質ゲート（ノイズ除去基準）の詳細

実際の追従サッカードを筋肉ノイズ、瞬目（まばたき）アーティファクト、頭部運動などから分離するため、検出された候補は**6つの厳格な品質ゲート（コード内で評価される正確な順序で記載）**によって検証されます。

### ゲート 1: 速度の外れ値チェック (Global)
* **Rule (ルール):** 検出されたすべての候補ピーク速度の四分位範囲（IQR）を算出します。ピーク速度が `中央値 + 3 × IQR` を超える極端な候補は除外されます。
* **生理学的根拠:** 本物の追従サッカードは一貫した速度範囲に収束します。極端な高速値は、筋肉のスパイク、電極ケーブルの揺れ、頭部運動といった生理学的に不可能なノイズによるものです。

### ゲート 2: 最小ステップ振幅チェック (150 ms)
* **Rule (ルール):** ピーク前ウインドウ（-170 ms 〜 -20 ms）とピーク後ウインドウ（+20 ms 〜 +170 ms）の間のEOG信号の絶対差（ステップ変化量）を計算します（過渡期を避けるため20 msの間隔を空けます）。このステップ振幅が `130 µV` 未満の場合、候補は除外されます。
* **生理学的根拠:** 隣接するフランキングLEDを追うための意図的な大域的眼球運動とは言えない、微小運動（マイクロサッカード）や高周波のベースラインドリフトを排除します。

### ゲート 3: EEGノイズ重複・アーティファクトチェック (-100 ms 〜 +300 ms)
* **Rule (ルール):** サッカード前後 -100 ms 〜 +300 ms のウインドウにおいて、EOG以外のすべてのEEGチャンネルのPeak-to-Peak振幅を調査します。いずれかのEEGチャンネルで振幅が `200 µV` を超えた場合、そのサッカードは除外されます。
* **生理学的根拠:** 体動や強い筋電位、電極の不安定さなど、全体的なノイズが発生している期間に起きたサッカードを排除し、後続の位相ロックEEG/SSVEP解析の精度低下を防ぎます。

### ゲート 4: 単一ピーク速度波形チェック (±100 ms)
* **Rule (ルール):** 候補ピークの前後 ±100 ms のウインドウ内で、支配的な速度ピークが1つだけ存在する必要があります。プライマリピークの高さの 40% を超えるセカンダリピークが検出された場合、その候補は除外されます。
* **生理学的根拠:** 本物の追従サッカードは、単一の急峻な変化を示す眼球運動です。速度プロファイルに複数のピークが存在する場合、視線追従の不安定さや瞬目の混入を示しています。

### ゲート 5: サッカード後の注視安定性チェック (200 ms)
* **Rule (ルール):** 速度ピークの直後 200 ms のウインドウにおいて、EOG信号の標準偏差（分散）がEOG全体の標準偏差の1.5倍未満でなければなりません。
* **生理学的根拠:** サッカードの後、目は即座にターゲット上に固定（注視）される必要があります。サッカード直後の高い分散は、目のドリフト、注視の不安定さ、または眼電位以外のノイズを示しています。

### ゲート 6: 波形の単調性（減速スロープ）チェック (300 ms)
* **Rule (ルール):** ピーク後 300 ms のウインドウにおけるEOG速度を25サンプルの移動平均で平滑化します。ピーク速度の5%にあたる不感帯（デッドバンド）を除外した上で、速度の符号反転（方向の変化）が4回以下である必要があります。
* **生理学的根拠:** サッカードの減速相は、新しい注視線レベルに向けてスムーズに速度ゼロに収束します。符号反転が頻繁に起こる場合、微小な震え（トレマー）やノイズ、追従の失敗を意味します。

## オンセット（開始点）計算とMATLABへの書き出し

MATLABスクリプトを生成する際、速度のピーク時刻をそのまま使うのではなく、各サッカードの正確な**オンセット（開始点）**を計算して出力します。
- **オンセット検出:** 速度ピーク位置から1サンプルずつ過去へ遡り、絶対速度がピーク値の **5%** 未満になるポイントを探索します（最大遡り幅は100 ms）。
- **レイテンシ変換:** 検出されたオンセットのサンプルインデックスを秒単位に変換し（`time = sample / sfreq`）、`latency = time * EEG.srate` として出力用MATLABスクリプトに記述します。
