# ローカル環境の構築と使用手順ガイド

東海大学 人間情報工学科 高尾研究室の研究リポジトリへようこそ。このガイドは、引き継ぎを受けた新しい研究メンバーが、自身のローカルPCにPython解析環境を迅速に構築し、各実験データの解析パイプラインを実行できるようにするための手順書です。

このリポジトリには、`src/python/` 以下に2つの主要な研究プロジェクトが含まれています。
1. **`led-flash`**: 青と緑の点滅光実験（メラノプシン・バイスタビリティ研究）における瞳孔径データ（Pupillometry）の解析。
2. **`phantom-array-experiment`**: ファントムアレイ効果研究における、`mne-python` を用いたEEG（脳波）およびEOG（眼球運動）データからのサッケード（急速眼球運動）自動検出とラベル付け。

## 1. 動作要件 (前提条件)

環境を構築する前に、PCに以下がインストールされていることを確認してください。
- **Python 3.8 以上** (Python 3.10以上を推奨)。ターミナルやコマンドプロンプトで以下のコマンドを実行し、バージョンを確認できます。
  ```bash
  python3 --version
  ```
- **Git** (リポジトリの複製やバージョン管理用)。

---

## 2. 自動スクリプトによる仮想環境の構築 (推奨)

ライブラリのバージョン競合を避け、依存関係をきれいに管理するため、Pythonの仮想環境（venv）を使用します。リポジトリのルートにある `requirements.txt` に、全プロジェクトで必要なライブラリがまとめられています。

macOS、Linux、Windowsの各OSで動作する自動構築スクリプトを提供しています。

### UNIX系 OS (macOS および Linux)
ターミナルを開き、リポジトリのルートディレクトリに移動して以下を実行します。
```bash
./setup_venv.sh
```
*※ 権限エラー（Permission Denied）が発生した場合は、先に `chmod +x setup_venv.sh` を実行して実行権限を付与してください。*

### Windows OS
コマンドプロンプト（cmd）または PowerShell を開き、リポジトリのルートディレクトリに移動して以下を実行します。
```cmd
setup_venv.bat
```

### Python経由で直接実行する場合 (OS共通)
スクリプトをPythonで直接実行することも可能です。
```bash
python setup_venv.py
```

### スクリプトが自動で実行する内容:
1. システムのPythonバージョンが 3.8以上 であるか確認します。
2. `.venv` という名前のディレクトリ内に仮想環境を作成します。
3. 仮想環境内の `pip` と `setuptools` を最新バージョンにアップグレードします。
4. `requirements.txt` に記載されたすべてのパッケージを自動でインストールします。

---

## 3. 手動による仮想環境の構築 (代替案)

スクリプトを使わず、手動でコマンドを実行して環境構築を行いたい場合は、以下の手順に従ってください。

1. **仮想環境の作成:**
   ```bash
   python -m venv .venv
   ```
2. **仮想環境の有効化 (アクティベーション):**
   - **macOS/Linux**: `source .venv/bin/activate`
   - **Windows コマンドプロンプト**: `.venv\Scripts\activate.bat`
   - **Windows PowerShell**: `.\.venv\Scripts\Activate.ps1`
3. **パッケージ管理ツールのアップグレード:**
   ```bash
   pip install --upgrade pip setuptools
   ```
4. **依存ライブラリのインストール:**
   ```bash
   pip install -r requirements.txt
   ```

## 4. 仮想環境の有効化と確認

環境構築が完了した後は、解析作業を行うためにターミナルを起動するたびに、**必ず仮想環境を有効化**する必要があります。

### 有効化コマンド:
- **macOS/Linux**:
  ```bash
  source .venv/bin/activate
  ```
- **Windows コマンドプロンプト**:
  ```cmd
  .venv\Scripts\activate.bat
  ```
- **Windows PowerShell**:
  ```powershell
  .\.venv\Scripts\Activate.ps1
  ```

有効化されると、プロンプトの先頭に `(.venv)` と表示されます。

### インストールの確認:
正しくインストールされたか確認するため、以下のコマンドを実行します。エラーが出なければ成功です。
```bash
python -c "import mne, pandas, numpy, scipy, matplotlib, japanize_matplotlib, pymatreader; print('ライブラリのインポートに成功しました！')"
```

## 6. トラブルシューティング

### MNE-Pythonの3Dビジュアライゼーションでのエラー
`phantom-array-experiment` では、3Dのセンサー配置図や頭部モデル描画のために `mne-qt-browser`、`pyqt6`、`pyvista`、`vtk` を使用します。
- OpenGLやディスプレイ表示に関連するエラー（特にリモートSSH、WSL、仮想マシン上で実行している場合）が発生した場合は、MNEの描画バックエンドを標準の2D matplotlibに切り替えることで解決します。
  ```python
  import mne
  mne.viz.set_browser_backend('matplotlib')
  ```

### グラフ内の日本語文字化けについて
グラフを描画する際、軸ラベルやタイトルに日本語を使用すると豆腐文字（□）化けすることがあります。
これを防ぐため、ノートブックには `japanize-matplotlib` が組み込まれています。新しいノートブックを作成した際は、プロットを行う前に必ず以下をインポートしてください。
```python
import japanize_matplotlib
```
